Ex 6
a)Ta biết:
Nếu người dùng nhấn Yes, option sẽ bằng 0.

Nếu người dùng nhấn No, option sẽ bằng 1.

Nếu người dùng nhấn Cancel, option sẽ bằng 2.

Nếu người dùng nhấn dấu X để đóng cửa sổ, option sẽ bằng -1.

=> Trong chương trình mà ta thực hiện, ta chỉ in ra Yes khi option = 0, 
nên khi người dùng chọn Cancel thì kết quả tương tự chọn No. Đều hiện ra
You've chosen: No
b)
Để custom thành Yes/No hoặc I do/I don't ta tạo nút riêng bằng mảng String,
rồi đưa cho showOptionDialog để nó hiển thị
Ví dụ code:
import javax.swing.JOptionPane;

public class chon {
    public static void main(String[] args) {

        String[] options = {"I do", "I don't"};

        int option = JOptionPane.showOptionDialog(null,
            "Do you want to change to the first class ticket?",
            "Confirm",
            JOptionPane.YES_NO_OPTION,
            JOptionPane.QUESTION_MESSAGE,
            null,
            options,
            options[0]);

        JOptionPane.showMessageDialog(null,
            "You've chosen: " + options[option]);
    }
}
